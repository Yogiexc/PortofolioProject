Portfolio Website - Bryan Yogie Saputra
Deskripsi Proyek
Proyek ini adalah website portofolio pribadi yang dibangun menggunakan HTML, CSS, dan Bootstrap. Website ini bertujuan untuk memperkenalkan diri saya, menampilkan keahlian dan proyek yang telah saya kerjakan, serta menyediakan cara untuk menghubungi saya.

Fitur Website
Website ini terdiri dari beberapa halaman yang dirancang untuk memberikan informasi yang jelas dan terstruktur mengenai diri saya dan karya-karya yang telah saya buat, yaitu:

Halaman Beranda (Home):

Menampilkan perkenalan singkat tentang saya.

Foto profil dan sambutan dari saya.

Halaman Tentang (About):

Memberikan deskripsi lebih mendalam tentang diri saya, termasuk latar belakang pendidikan, hobi, dan minat saya.

Halaman Portofolio (Portfolio):

Menampilkan proyek atau tugas yang telah saya kerjakan. Setiap proyek mencakup gambar, judul, dan deskripsi yang menjelaskan proyek tersebut.

Halaman Kontak (Contact):

Berisi formulir untuk pengunjung yang ingin menghubungi saya. Formulir ini memungkinkan pengunjung untuk mengirimkan nama, email, subjek, dan pesan.

Struktur File
Berikut adalah struktur file dan folder yang ada pada proyek ini:


/project-directory
│
├── index.html          # Halaman utama (Home)
├── about.html          # Halaman tentang (About)
├── portfolio.html      # Halaman portofolio (Portfolio)
├── contact.html        # Halaman kontak (Contact)
│
├── /css                # Folder untuk file CSS
│   └── style.css       # File CSS utama untuk desain website
│
├── /img                # Folder untuk aset gambar
│   ├── profile.jpg     # Foto profil saya
│   └── project1.jpg    # Gambar proyek 1
│
└── README.txt          # Deskripsi proyek ini


Cara Menjalankan Website
Untuk melihat website ini, cukup ikuti langkah-langkah berikut:

Download atau clone repositori ini ke komputer Anda.

Buka folder yang berisi file-file HTML dan CSS.

Buka file index.html menggunakan browser favorit Anda.

Tidak diperlukan konfigurasi atau server khusus untuk menjalankan website ini karena semuanya sudah berbasis HTML dan CSS.

Catatan Pengembangan
Proyek ini sudah berjalan dengan baik, namun ada beberapa bagian yang masih dalam tahap pengembangan atau perbaikan, di antaranya:

Responsiveness: Website ini sudah responsive pada beberapa ukuran layar, namun beberapa elemen mungkin perlu penyesuaian lebih lanjut untuk tampilan pada perangkat mobile.

Konten Portofolio: Beberapa proyek pada halaman portofolio masih dalam tahap pengisian dan dapat diperbarui lebih lanjut.

Masalah yang Dihadapi
Tidak ada masalah besar yang ditemukan sejauh ini. Namun, beberapa fitur seperti formulir kontak belum terhubung dengan backend untuk mengirim email atau pesan. Ini masih membutuhkan pengaturan tambahan dengan server atau API untuk pengiriman data.
