Penjelasan Proyek
Proyek ini adalah website portofolio pribadi yang dibangun menggunakan HTML, CSS, dan Bootstrap. Website ini bertujuan untuk memperkenalkan diri saya, menampilkan keahlian dan proyek yang telah saya kerjakan, serta menyediakan cara untuk menghubungi saya.

Website ini terdiri dari beberapa halaman:

Halaman Beranda (Home): Menampilkan perkenalan singkat tentang saya, foto profil, dan sambutan.

Halaman Tentang (About): Menyediakan deskripsi lebih dalam tentang diri saya, termasuk pendidikan, hobi, dan minat saya.

Halaman Portofolio (Portfolio): Menampilkan minimal tiga proyek atau tugas yang telah saya kerjakan, beserta gambar, judul, dan deskripsi masing-masing.

Halaman Kontak (Contact): Berisi formulir kontak yang memungkinkan pengunjung untuk mengirimkan nama, email, subjek, dan pesan.
